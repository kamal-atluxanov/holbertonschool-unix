# Unix Basics

This repository contains notes and resources for learning **Unix operating system fundamentals** and basic command-line usage.


##  What is Unix?

Unix is a **multiuser, multitasking operating system** widely used in servers, workstations, and embedded systems. 
It provides a **powerful command-line interface** and a stable environment for software development and system administration.


##  Key Concepts

### 1. **File System**
- Hierarchical structure starting from root `/` 
- Directories, files, and symbolic links 
- Commands: `ls`, `cd`, `pwd`, `mkdir`, `rm`

### 2. **Permissions & Ownership**
- Control access using read (`r`), write (`w`), execute (`x`) permissions 
- Commands: `chmod`, `chown`, `chgrp`

### 3. **Processes & Jobs**
- Manage running programs and background jobs 
- Commands: `ps`, `top`, `kill`, `jobs`, `fg`, `bg`

### 4. **Shell & Scripting**
- Shells like **Bash, Zsh** provide command-line interface 
- Automate tasks with shell scripts

### 5. **Networking & Utilities**
- Commands: `ping`, `netstat`, `ssh`, `scp` 
- Monitor network and transfer files securely

### 6. **Package Management**
- Install and manage software 
- Examples: `apt`, `yum`, `pkg`, depending on Unix variant


##  Learning Goals

- Navigate and manage the Unix file system 
- Control file permissions and processes 
- Automate repetitive tasks with shell scripting 
- Use basic networking commands and utilities 


**Tip:** Start by exploring the terminal, learning basic commands, and then move to shell scripting and process management.
